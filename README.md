# A Frog Journey
## Description du Projet

Dans le cadre d'un projet, "A Frog Journey", a été développé en utilisant les Windows Form Applications. Ce jeu met en scène une grenouille capable de sauter, de se déplacer à gauche et à droite, dans le but de ramasser des fruits pour augmenter son score à travers trois niveaux. Le premier niveau est le plus simple et le troisième est le plus difficile. Le joueur peut passer d’un niveau à un autre en franchissant des portes.
![afrogjourney](https://github.com/thomasiafrate1/Iafrate_Thomas_WFA/assets/113009615/edf99c86-044f-497e-8db5-e4683e24d10e)

## Attentes du Client

Le client avait plusieurs thèmes à choisir: un platformer, un snake, un space invader, un casse-brique, un pong, ou un thème au choix. "A Frog Journey" a été conçu en tant que plateformer pour répondre à ces attentes et consignes du client.

## Langages et Outils

Le projet a été réalisé en utilisant le langage de programmation C# et la technologie Windows Form pour l'interface graphique. Le développement a été effectué sur Visual Studio 2022 pour permettre de construire un logiciel stable.

    Langage de Programmation: C#
    Technologie d'Interface Graphique: Windows Form
    Environnement de Développement: Visual Studio 2022
